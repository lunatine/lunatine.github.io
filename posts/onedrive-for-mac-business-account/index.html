[Mac] OneDrive 비지니스 계정 추가하기

얼마전까지만 해도 One Drive 비지니스 버전의 Mac OS X Preview 버전이 제공되고 있었는데 잦은 크래시 및 여러 문제가 있었다. (개인적인 경험일 수 있음) 그러다가 최근 정식 릴리즈를 위해서 Preview 링크가 사라졌는데 현재 정식으로 서비스 중에 있는 일반 One Drive 클라이언트를 통해서 비지니스 계정을 추가하는 방법을 공유한다.

OneDrive 다운로드

먼저, App Store에서 onedrive를 검색해서 어플리케이션을 설치한다.

그리고, 기존에 MS OneDrive 계정 (일반계정)이 있는 경우와 없는 경우에 따라서 방법이 조금 다른데 일반 계정이 있는 경우(현재 사용 중인 경우)는 2)번 항목을 참고하고 없는 경우나 비지니스 계정만을 사용하고 싶을 때는 1)번 항목을 참고하자.

1) MS OneDrive 일반 계정이 없는 경우

  • MS OneDrive 일반 계정이 없거나 혹은 비지니스 계정만 사용하기를 원할 경우에 해당한다.

터미널 프로그램을 실행하고나서 아래 내용을 입력한다.

$ defaults write com.microsoft.OneDrive-mac DefaultToBusinessFRE -bool True

위 명령으로 설정되는 파라미터는 OneDrive의 기본 계정을 비지니스 계정으로 설정한다는 의미이다. 위 명령을 실행한 후에 다시 아래 내용을 입력한다.

$ defaults write com.microsoft.OneDrive-mac EnableAddAccounts -bool True

위 명령은 계정을 추가를 활성화하는 것을 의미한다.

이렇게 명령을 입력하고 실행하게 되면 아래와 같은 화면들을 만날 수 있으며 로그인 후에 원하는 위치에 OneDrive 폴더를 지정하면 된다.

2) MS OneDrive 일반 계정이 있는 경우

만약, 이미 MS OndDrive 일반 계정이 있고 사용중인 경우에는 아래 내용을 입력하여 계정 추가 기능을 활성화 한다.

$ defaults write com.microsoft.OneDrive-mac EnableAddAccounts -bool True

위 명령을 입력하고 나서 OneDrive의 기본 설정... 메뉴를 실행하여 계정 탭으로 들어가서 아래와 같이 비지니스 계정 추가 버튼을 클릭하면 기존 OneDrive 외에 별도의 OneDrive 프로그램이 실행되며 로그인 창이 뜨게 된다.

새로 뜬 로그인창에 비지니스 계정으로 로그인하여 원하는 디렉토리를 선택하고 나면 아래와 같이 2개의 MS OneDrive 아이콘을 확인 할 수 있다. 즉, 2개의 OneDrive가 일반 계정과 비지니스 계정으로 따로 동작한다.

마무리

아마도, 정식버전이 나오게 되면 위 방법은 의미 없을 것으로 보인다.
