검색어에 대한 구글 센스

전 세계적으로 사랑받는 크롬브라우저에서 특정 검색 키워드를 사용하면 키워드의 의미를 직접 보여주는 방식으로 동작한다. 물론 크롬만 되는 것은 아니고 HTML5를 지원하는 모든 브라우저에서 사용이 가능하다. HTML5를 지원하는 브라우저에서 구글검색으로 들어가서 아래의 키워드를 사용하면 된다. tilt 또는 askew : 화면이 기울어진다. merry christmas : 성탄절 장식이 등장 let it snow : 화면에서 눈이 내리고 화면을 덮기시작 하며 마우스 클릭&드래그로 눈을 지울 수 있다 do a barrel roll : 화면이 데굴데굴 구른다 ascii art : 아는 사람은 아는 글자로 그림을 표현하는 방식으로 검색결과가 바뀐다....
