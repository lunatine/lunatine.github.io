yum 저장소에서 checksum error 발생

yum checksum error 증상 yum 저장소를 새로 생성하였으나 해당 목록을 받아올 때 아래 메시지 발생 Loaded plugins: security RHEL5.10/primary | 1.2 MB 00:00 http://exampledomain.com/rhel/5Server/x86_64/5.10/repodata/primary.xml.gz: [Errno -3] Error performing checksum Trying other mirror. 발생 조건 RHEL5 저장소를 RHEL6에서 생성하였을 경우 원인 및 해결방안 createrepo로 저장소를 생성 할 때 저장소 정보를 RHEL5의 Checksum은 SHA-1 기반으로 되어있는 반면에 RHEL6의 경우는 SHA-256으로 생성하기 때문에 RHEL5에서 해당 저장소를 사용하려고 하면 오류가 발생하는 것이다....
